# Product Overview

Arcadia is a modern library management system (LMS) and bookstore platform designed for small to medium-sized libraries, school libraries, community libraries, and bookstores.

## Core Features

- **Book Management**: Comprehensive cataloging with title, author, ISBN, category, publisher, description, price, stock, and metadata. Supports bulk import/export via CSV.
- **Member Management**: Track members, contact information, and borrowing history.
- **Circulation System**: Automated check-in/check-out with due date calculation and overdue tracking.
- **Bookstore Mode**: Checkout process, inventory management, and sales tracking.
- **Search & Filtering**: Advanced search across books and members.
- **Analytics Dashboard**: Real-time insights on popular books, lending trends, and usage statistics.
- **Role-Based Access**: Separate dashboards for library staff and bookstore owners.

## User Roles

- **Library Staff**: Manage circulation, members, and library operations.
- **Bookstore Owners**: Handle sales, inventory, and customer transactions.
- **Members/Customers**: Browse catalog and view their borrowing/purchase history.

## Key Differentiators

- Modern, responsive UI that works on desktop, tablet, and mobile
- Real-time updates and notifications
- Secure authentication with row-level security
- Open-source and customizable
